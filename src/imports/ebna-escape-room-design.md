Create a comprehensive UI/UX design system and interactive prototype for "EBNA Escape Room" — a digital gamified learning platform for nursing students focused on evidence-based practice.

CORE CONCEPT:
Transform an analog nursing escape room into an immersive digital experience where students solve clinical puzzles, analyze research studies, and apply evidence-based knowledge through cooperative, scenario-based challenges.

DESIGN REQUIREMENTS:

1. VISUAL IDENTITY:

- Theme: Medical mystery meets modern digital interface
- Color palette: Clean clinical whites, calming teals/medical blues, alert oranges for urgency/puzzles, success greens
- Typography: Highly legible sans-serif for clinical data, monospace for "system/terminal" elements
- Style: Blend professional healthcare aesthetics with engaging game UI elements (progress bars, badges, timers)

2. KEY SCREENS TO DESIGN:

- Landing/Login: Immersive entry with "Mission Briefing" style onboarding
- Dashboard: Student hub showing available cases, progress stats, team formation
- Escape Room Interface: Main game screen with:
  - Scenario narrative panel (patient case introduction)
  - Interactive evidence board (drag-drop studies, highlight key findings)
  - Puzzle modules (clinical calculators, PICO analysis tools, study appraisal checklists)
  - Team collaboration panel (chat, shared notes, role assignments)
  - Progress tracker (visual "room escape" metaphor - unlocking doors/sections)
- Evidence Analysis Workspace: Split-screen study viewer with annotation tools
- Debrief/Reflection Screen: Post-case summary, learning outcomes, peer comparison

3. INTERACTION PATTERNS:

- Gamification elements: XP points, badges (Critical Thinker, Evidence Hunter, Team Player), leaderboards
- Time-pressure indicators without causing anxiety (optional timer modes)
- Collaborative cursors/annotations for team-based analysis
- Haptic/visual feedback for correct diagnoses or successful evidence applications
- Branching scenarios based on student choices

4. ACCESSIBILITY & UX:

- High contrast modes for clinical environments
- Mobile-responsive for tablet-based bedside/ward simulations
- Clear wayfinding—students should always know their objective
- Error-friendly design: Wrong answers trigger learning hints, not just failures

5. CONTENT MODULES TO VISUALIZE:

- PICO Framework interactive builder
- Study quality appraisal checklists (CASPed tools)
- Clinical scenario simulations with decision trees
- "Evidence Lock" puzzles requiring correct statistical interpretation to unlock next clues

DELIVERABLES:

- Design system (colors, typography, components)
- High-fidelity mockups of all key screens
- Interactive prototype showing student journey from login to case completion
- Component library for nursing education patterns (vital signs displays, chart interfaces, research article viewers)

TONE: Professional yet engaging, clinically credible but not intimidating, intellectually challenging yet accessible to nursing students at various levels.
